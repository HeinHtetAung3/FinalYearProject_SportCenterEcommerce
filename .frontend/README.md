# Sports E-commerce Frontend Foundation

## Run locally
1. Install dependencies: `npm install`
2. Copy `.env.local` and adjust values.
3. Start app: `npm run dev`

## Implemented in phase1-foundation
- React + Vite + Tailwind foundation.
- Shared app layout with Header/Footer and route shell.
- Error boundary and loading UI primitives.
- `AuthContext` and `CartContext` with localStorage persistence.
- Axios client with token injection, refresh handling, and normalized API errors.
